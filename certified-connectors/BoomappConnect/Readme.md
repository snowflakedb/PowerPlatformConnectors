# BoomApp Connect

Boost customer engagement with Boomerang’s SMS, Voice, and Email Connector for the Microsoft Power Platform. Deliver automated alerts, reminders, and two-way conversational messaging directly from your Microsoft applications. Leverage real-time responses to streamline and automate communication workflows. With advanced inbound messaging features, customers can initiate conversations that integrate instantly into your business processes, driving faster, more efficient interactions.

## Pre-requisites
You will need the following to proceed:
* A Microsoft Power Automate plan with Custom Connector feature
* A Boomerang Subscription
* Boomerang API authentication details.
